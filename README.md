# BluFizz
Blufizz is a tool to store the scanned bluetooth and wifi devices. A database keep each device details,
time statistics and gps localisation.

# Current functions
- Bluetooth BR/EDR and LE scan
- Store scan results in a database

# Limitations
Works better with public address, device with random address can be recorded multiple times.
